## Couverture

## 1. Qui ?
EisenFlow a été créé pour aider les personnes qui reçoivent beaucoup d’emails et qui ont du mal à tout gérer. Le but est de réduire le stress et de mieux organiser le travail grâce à l’Intelligence Artificielle.

## 3. Description de l'application
Notre application permet à l'utilisateur de charger ses mails dans notre application. Cette application va ensuite donner la possibilité :
* Soit de **classer automatiquement** les mails dans la **Matrice Eisenhower** pour que les mails se fassent taguer.
* Soit de laisser classer et taguer les mails **manuellement** par l'utilisateur.

Parmi les fonctionnalités présentes dans notre application, il y a la possibilité d'envoyer les résultats dans Gmail directement, classer et taguer les mails. Dans les tags disponibles, il y a le **DO**, **PLAN**, **DELEGATE** et **DELETE**. Le DO est le tag qui...



## 4. Analyse de Concurrence
Parmi les concurrents de notre application, on retrouve plusieurs applications largement utilisées qui proposent des fonctionnalités similaires :

* **Boîtes mails classiques (Gmail, Outlook, AppleMail) :** Elles permettent la gestion centralisée, la création de dossiers ou de règles manuelles. Contrairement à notre application, elles visent à faciliter la communication mais ne permettent pas encore le tri automatique intelligent.
* **Boîtes mails avec IA (SuperHuman) :** Elles peuvent résumer un mail et écrire une réponse. SuperHuman ressemble à notre application (IA + réponse automatique de type DELEGATE), mais contrairement à nous, SuperHuman n'utilise pas une **IA locale** mais envoie les données sur des serveurs en ligne, ce qui pose une problématique de confidentialité.
* **Gestion de projet (Jira) :** Jira propose des tableaux Kanban très utiles pour le travail en équipe. Notre application permet de suivre les tâches directement extraites des mails, contrairement à Jira où chaque tâche doit être saisie manuellement.
